# dotfiles
Containing dotfiles with aliases and shorcuts

## List of aliases for git bash

```sh
alias vue='winpty vue.cmd'
alias ll='ls -la'
alias ..='cd ..'
alias ~='cd ~'

alias g='git'
alias gs='git status'
alias ga='git add'
alias gaa='git add -A'
alias gc='git commit -m "Update application"'
alias gd='git diff'
alias gp='git push'
alias gpll='git pull'
alias gb='git branch'
alias gst='git stash'
alias gsta='git stash apply'
```
